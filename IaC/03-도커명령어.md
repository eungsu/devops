# 도커 명령어

## 도커의 주요 명령어
```shell
### 도커 버전 확인
$ docker version

### 도커이미지 내려받기
### docker pull 도커이미지이름
$ docker pull httpd

### 도커이미지 목록 조회하기
$ docker images
$ docker image ls

### 도커이미지 삭제하기
### 실행중인 컨테이너의 도커 이미지는 삭제할 수 없다.
### docker image rm 도커이미지이름

### 도커 컨테이너 실행하기
### docker run 도커이미지이름
$ docker run hello-world
### docker run --name 컨테이너이름 도커이미지이름
$ docker run --name hello hello-world
### docker run -d -p 호스트포트번호:컨테이너포트번호 --name 컨테이너이름 도커이미지이름
$ docker run -d -p 8080:80 --name web-server httpd
### docker run -d -p 호스트포트번호:컨테이너포트번호 -v 볼륨이름:컨테이너경로 --name 컨테이너이름 도커이미지이름
$ docker run -d -p 8080:80 -v web:/usr/local/apache2/htdocs --name web-server httpd
### docker run -d -p 호스트포트번호:컨테이너포트번호 -v 볼륨이름:컨테이너경로 -e 환경변수이름=환경변수값 --name 컨테이너이름 도커이미지이름
$ docker run -d \
  -v /my/own/datadir:/var/lib/mysql \
  –e MYSQL_ROOT_PASSWORD=1234 \
  –e MYSQL_DATABASE=testdb \
  –e MYSQL_USER=user1 \
  -e MYSQL_PASSWORD=1234 \
  --name 
```
