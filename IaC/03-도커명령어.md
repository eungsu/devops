# 도커 명령어

## 도커의 주요 명령어
```shell
### 도커 버전 확인
$ docker version

### 도커이미지 내려받기
### docker pull 도커이미지이름
$ docker pull httpd

### 도커이미지 목록 조회하기
$ docker images
$ docker image ls

### 도커이미지 삭제하기
### 실행중인 컨테이너의 도커 이미지는 삭제할 수 없다.
### docker image rm 도커이미지이름

### 도커 컨테이너 실행하기
### docker run 도커이미지이름
### docker run --name 컨테이너이름 도커이미지이름
### docker run -d --name 컨테이너이름 도커이미지이름
### docker run -d -p 호스트포트번호:컨테이너포트번호 --name 컨테이너이름 도커이미지이름
### docker run -d -p 호스트포트번호:컨테이너포트번호 -v 볼륨이름:컨테이너경로 --name 컨테이너이름 도커이미지이름
### docker run -d -p 호스트포트번호:컨테이너포트번호 -v 볼륨이름:컨테이너경로 -e 환경변수이름=환경변수값 --name 컨테이너이름 도커이미지이름

``-
