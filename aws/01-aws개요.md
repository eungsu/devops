# 클라우드 컴퓨팅

## 클라우드 컴퓨팅이란?
클라우드 컴퓨팅은 IT 리소스를 인터넷을 통해 **온디맨드**로 제공하고 **사용한 만큼만 비용**을 지불하는 것이다.

## 클라우드 컴퓨팅의 이점
- 확장성 및 유연성
  + 비즈니스 수요에 맞게 리소스와 스토리지를 빠르게 확장할 수 있다.
  + 리소스를 사용하지 않을 때 빠르게 규모를 축소할 수 있다.
- 비용 절감
  + 실제로 사용하는 리소스에 대한 비용만 지불한다. 
- 더욱 편리해진 공동작업
  + 클라우드 스토리지를 사용하면 언제 어디서나 필요한 데이터를 사용할 수 있다.
  + 사용자는 위치나 특정 기기에 얽매이지 않고 전 세계 어디서난 어느 기기에서든 인터넷에 연결되어 있다면 데이터에 엑세스 할 수 있다.
- 고급 보안
  + 클라우드 컴퓨팅은 다양하고 심층적인 보안 기능, 자동 유지보수, 중앙 집중식 관리 덕분에 보안 상태를 강화할 수 있다.
  + 클라우드 제공업체는 최고의 보안 전문가를 고용하고 최첨단 솔루션을 채택하여 보다 강력한 보호를 제공한다.
- 데이터 손실 방지
  + 클라우드 제공업체는 백업 및 재해 복구 기능을 제공한다.
  + 클라우드에 데이터를 저장하면 하드웨어 오작동, 악의적인 위협과 같은 긴급 상황에서나 간단한 사용자 오류 발생 시에도 데이터 손실을 방지할 수 있다.

## 클라우드 컴퓨팅 모델
- **IaaS(Infrastructure as a Service)***
  + 인프라만 제공
  + OS를 직접 설치하고 필요한 소프트웨어를 개발해서 사용
  + 가상의 컴퓨터를 임대하는 것과 비슷
  + 예) AWS EC2
- **PaaS(Platform as a Service)**
  +  인프라 + OS + 프로그램 실행에 필요한 싫행환경
  +  개발한 어플리케이션 코드를 올려서 바로 실행할 수 있도록 구성
  +  예) Firebase, Google App Engine 등
- **SaaS(Service as Service)**
  + 인프라 + OS + 서비를 제공하는 소프트웨어
  + 서비스 자체를 제공
  + 별다른 설정 없이 서비스만 이용
  + 예) Gmail, DropBox, Slack, Google Docs
